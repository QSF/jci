Instalar o GIT
Gerar o SSH https://github.com/settings/ssh
Copiar a chave SSH publica gerada para o github
Clona o Repositorio https://github.com/QSF/jci para o seu htdocs (git clone git@github.com:QSF/jci.git) 
Criar o branch local develop (git checkout -b develop origin/develop) => http://stackoverflow.com/questions/67699/how-do-i-clone-all-remote-branches-with-git
Verificar se está trabalhando no branch certo (git branch, caso nao esteja da um git checkout nome_do_branch)
Criar ou Editar os arquivos
Manda os arquivos para o Stage => git add .
Comita => git commit -m "mensagem"
Manda pro repositório github => git push origin develop
Atualizando seu branch local com o do  repositório => git pull origin develop
Para ver os commits dar git log
